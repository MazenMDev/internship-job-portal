## Overview

JobConnect is a full-stack job portal platform that bridges job seekers with employers. The platform features dual user interfaces for job seekers and companies, advanced search capabilities, real-time notifications, and a comprehensive admin dashboard. Built with PHP, MySQL, and modern JavaScript, it offers a responsive design with dark/light theme support.

## Features

### Job Seekers

-   Advanced job search with filters (category, location, salary, type)
-   Job bookmarking and application tracking
-   Profile management with CV upload
-   Real-time application status notifications

### Companies

-   Post and manage job listings
-   Application review and status updates
-   Company profile and branding
-   Dashboard with analytics

### Administrators

-   Platform oversight and user management
-   Company verification system
-   Content moderation
-   Analytics dashboard
